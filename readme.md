## Houdini
```
Houdini > radial_menu_display_options.py
```

A pie menu to toggle the different display options (e.g. Display points, Display point normals, etc.). It also works 
with custom markers. 

![Screenshot showing the display options pie menu](_public/.github/images/screenshot_display_piemenu.png)  

### Installation:

1. Open Houdini
2. Edit > Radial Menus...
3. Add menu
4. Set a name and label
5. Paste the code
